# EDC Tracker

**Version**: 1.000070

---

## ✨ Short Description
A seamless system built to keep records and track attendance in real-time.

---

## 📋 App Overview
**Category**: Business Solution  
**Function**: Quality Management  
**Industry**: Non-Profit  

This app is designed to help organizations, such as churches or nonprofits, monitor attendance effectively and with ease. It leverages **Google Sheets**, **AppScript** for backend automation, **QR Code generation** for easy tracking, and **Google Drive** and **Gmail** for email notifications and file management. The project is hosted on **Google Cloud**, ensuring scalability and reliability.

---

## 🌟 Features
- ✅ Real-time attendance tracking  
- 📊 Easy data CRUD and storage  
- 📝 Seamless data collection with JotForm  
- 📲 Quick attendance logging with QR Codes  
- 🔐 Secure client data handling  
- ☁️ Scalable cloud hosting on Google Cloud  

---

## 🛠 Technical Details
- **📱 AppSheet**: App interface and functionality  
- **⚙️ Google AppScript**: Automation and custom scripts  
- **📊 Google Sheets**: Data storage and management  
- **📝 JotForm**: Form data collection  
- **📲 QR Code Generation**: For easy attendance logging  
- **📂 Google Drive**: Stores templates and user-generated files  
- **📧 Gmail**: Sends email notifications  
- **📊 Google Slides**: Custom templates for dynamic reporting  
- **☁️ Google Cloud**: Hosting for high availability and scalability

---

## 🎯 Purpose and Target Users
This app is designed with non-profits and faith-based organizations in mind. Its primary purpose is to simplify attendance tracking, saving time and reducing manual errors. It’s scalable for broader use in similar organizations.

---

## 👩‍💻 User Guidance
Users can easily navigate the app to log attendance and access real-time data. For form-based data collection, users can utilize the integrated JotForm. For additional support, refer to the provided documentation.

## 🔒 Privacy and Terms of Use
- **Privacy Policy**: Users' data is handled securely and kept safe.  
- **Terms of Use**: This app is designed for organizational use and adheres to a clear set of user agreements.

## 📚 Documentation
### 📝 Design Details
The app uses a **Google Sheets** backend, with some fields populated through **AppScript**. This ensures data accuracy and supports the app’s real-time capabilities. **JotForm** is integrated for seamless form data collection.

For additional details, visit the [documentation page](https://www.appsheet.com/template/appdoc?appId=90b3f6db-c32a-4c23-bc6f-68e788da4b08).
